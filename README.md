# Data-BIT-258994056-
This repository contains public materials for the study, “Predicting Risk-Aversion Maneuvers in Pre-Crash Scenarios Based on Diverse Driving Objectives.”

A full-structure sub-dataset was created by randomly sampling 50% of the observations from each of the four combinations of driving objective and action (Proactive/Reactive and Steering/Braking), resulting in 199 observations. The analytical pipeline and sub-dataset are provided to demonstrate the analytical workflow and facilitate verification by future researchers.
